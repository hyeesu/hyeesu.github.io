1. cd (change directory)
<br>
• 경로 이동
<br>
• 절대 경로와 상대 경로 이동 가능

    ```terminal
    $ cd documents
    ```

          <th>의미</th> 
      </tr>
      </thead>
      <tbody>
        <tr>
          <th>cd directory</th>
          <th>directory 이동 (절대 경로)</th>
        </tr>         
        <tr>
          <th>cd /</th>
          <th>root directory 이동</th>
        </tr>             
        <tr>
          <th>cd ..</th>
          <th>상위 directory 이동 (상대 경로)</th>
        </tr>       
        <tr>
          <th>cd -</th>
          <th>이전 directory 이동 (뒤로가기와 같음..)</th>
        </tr> 
        <tr>
          <th>cd 또는 cd ~</th>
          <th>home directory 이동</th>
        </tr>                           
      </tbody>
    </table>  
      
2. pwd (print working directory)
<br>
• 현재 디렉터리 절대 경로 표시

    ```terminal
    $ cd home
    $ pwd

    /home
    ```

3. ls (list directory contents)
<br>
• 현재 위치한 디렉터리에 있는 디렉터리나 파일 리스트 출력
<br>
• 디렉터리를 지정하여 목록 출력 가능
<br>
• 옵션 미사용 시 숨김 파일 보이지 않음
<br>
• 옵션을 혼합하여 사용할 수 있음

    ```terminal
    $ ls /home
    $ ls -al
    $ ls -ef
    $ ls

    mysql tomcat oracle
    ```

4. head
<br>
• 파일의 앞부분 내용을 지정한 행의 수 만큼 출력
<br>
• 옵션을 지정하지 않으면 default 10줄

    ```terminal
    $ head 파일
    $ head -n 20
    $ head -c 100
    $ head 기존파일 > 생성파일
    ```

    <table class="table table-striped table-bordered">
      <thead>
        <tr>
          <th>명령어, 옵션</th>
          <th>의미</th> 
        </tr>
      </thead>
      <tbody>
        <tr>
          <th>-c {number}</th>
          <th>지정 킬로바이트 단위 출력</th>
        </tr>         
        <tr>
          <th>-n {number}</th>
          <th>지정 라인 단위 출력</th>
        </tr>                
        <tr>
          <th>-q</th>
          <th>파일 이름 header 미출력</th>
        </tr>       
        <tr>
          <th>-v</th>
          <th>파일 이름 header 출력</th>
        </tr> 
        <tr>
          <th>기존파일 > 생성파일</th>
          <th>옵션에 따른 출력 결과를 생성파일로 생성</th>
        </tr>                           
      </tbody>
    </table>

5. tail
<br>
• 파일의 뒷부분 내용을 지정한 행의 수 만큼 출력
<br>
• 옵션을 지정하지 않으면 default 10줄

    ```terminal
    $ tail 파일
    $ tail -n 20
    $ tail -c 100
    $ tail 기존파일 > 생성파일
    ```

6. grep
<br>
• 특정 파일에서 지정한 문자열이나 정규표현식을 포함한 행 출력
<br>
• tail 또는 ls 등 조합하여 응용되는 경우가 많음

    ```terminal
    $ grep 'error' 파일명
    $ grep 'error' *
    $ grep 'error' *.log
    ```
